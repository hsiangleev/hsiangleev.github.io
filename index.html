gitbook安装使用

1. nodejs，npm安装

2. 淘宝镜像
npm install -g cnpm --registry=https://registry.npm.taobao.org

3. gitbook安装
cnpm install gitbook gitbook-cli -g

4. 初始化目录
gitbook init ./docs

5. 新建文件book.json

6. 运行服务

  1. gitbook serve 运行服务并生成静态文件
  2. gitbook build 只生成静态文件

7. 浏览器打开 http://localhost:4000/

8. 添加章节

  • 在目录 docs/SUMMARY.md 中添加
  • 在目录 docs/ 下新建文件夹 chapter1 ，并在该文件夹下新建文件cascader.md 即可

9. 导出pdf

  • 下载并安装 calibre - E-book
  • 执行命令 gitbook pdf ./ ./document.pdf 即在当前目录导出

10. 修改导出的样式

  • book.json 添加代码
